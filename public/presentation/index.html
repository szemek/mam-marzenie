Introduction

  1. Mam Marzenie is an NGO aspiring to make the dreams of heavily ill children come true.
  2. Mainly problems are communication between donors and organization.
  3. Create a tool supporting the communication between the Foundation and its Donors.

Design

  1. Open sources technologies: Ruby on Rails
  2. Features:
    • Volunteers can create and edit children dreams.
    • Volunteers can track realization of dreams.
    • Volunteers can manage sponsors information.
    • Volunteers can generate presentation for meetings.
    • Volunteers can search similar dreams of others.
